## Code de conduite

En participant à ce projet, vous acceptez de respecter un environnement accueillant et respectueux pour tous. Soyez courtois et constructif dans vos échanges.

## Comment contribuer

### Signaler un bug

1. Vérifiez que le bug n'a pas déjà été signalé dans les [Issues](https://github.com/ebigeard/pyUPSTIlatex/issues)
2. Créez une nouvelle issue avec le template "Bug Report"
3. Fournissez le maximum d'informations :
   - Version de pyUPSTIlatex
   - Version de Python
   - Système d'exploitation
   - Commande exécutée
   - Message d'erreur complet
   - Fichier `.tex` minimal reproduisant le problème (si possible)

### Proposer une fonctionnalité

1. Vérifiez dans les [Issues](https://github.com/ebigeard/pyUPSTIlatex/issues) et [Discussions](https://github.com/ebigeard/pyUPSTIlatex/discussions) si la fonctionnalité n'a pas déjà été proposée
2. Créez une nouvelle issue avec le template "Feature Request"
3. Décrivez clairement :
   - Le problème que cela résout
   - Comment vous imaginez la solution
   - Des exemples d'utilisation
